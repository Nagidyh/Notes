基于企业微信小程序，构建的一套学校社团管理系统。

学生、家长通过企微程序报名、参与社团，学校、老师通过后台管理社团。



### 小程序端



#### 企微小程序 前端

常规小程序搭建，请求推送到由 eggjs 搭建的 serverless 中进行业务操作。

#### eggJs serverless API 支持

基于 eggjs 搭建的 serverless 服务支持，提供小程序需要使用的 API 逻辑。

#### iview 2 管理前端



#### thinkphp 6 API 支持
